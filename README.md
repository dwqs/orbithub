## Tools
[crx](https://github.com/oncletom/crx) is a node.js command line app for packing Google Chrome extensions.

```
npm i crx -g
crx crx pack -o outputPath
```

>`outputPath` is like `/xxx/orbithub.crx`, it must include the suffix .crx
